SEG3125 - Design and Analysis of User Interface

Below are labs conducted throughout the course along with a short description





Lab Description
Lab 1 Built a static web page that lays out a survey to evaluate the user interface of a site. Technologies used include HTML and CSS.
Lab 2 Built a dynamic mini-website allowing a user to purchase items in an online grocery store. Technologies used include HTML, CSS, and JavaScript
Lab 3 Built an alternative version for the website built during Lab 2 using UI implementation toolbox. Technologies used include HTML, CSS, and JavaScript
Lab 4 Built a semi-static site for a barbershop business that respects the design principles presented in the book Design of Everyday Things by Dr. Norman. Technologies used include HTML, CSS, Bootstrap and JavaScript
